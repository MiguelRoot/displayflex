Methods

Calculate the sizes that you indicate.

.myNameClass {
  @include row(100px 1fr 100px);
}
<div class="row myNameClass">
    <div class="col">
        <div class="box"> </div>
    </div>
    <div class="col">
        <div class="box"> </div>
    </div>
    <div class="col">
        <div class="box"> </div>
    </div>
</div>

fit

Calculate the width and height that grow in proportion.

.myNameFit {
  @include fit(273px 50px);
}
<div class="myNameFit">
    <div class="box"></div>
</div>

query

Mixin that indicates from

.myclassFrom {
    font-size: 12px;
    @include from(md) {
        font-size: 16px;
    }
}

Mixin that indicates until

.myclassFrom {
    font-size: 16px;
    @include until(sm) {
        font-size: 12px;
    }
}
