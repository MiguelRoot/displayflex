# DisplayFlex
> Flexible based customizable grid. [documentation](https://miguelroot.github.io/displayflex/)


## Install:

### npm
``npm i displayflex``

### import
```
@import "node_modules/displayflex/displayflex";
```
## config
```
$dev: false;
$rem: 16px;

$columns: (
    "breakpoints": (
        "xs": 0,
        "sm": 769px,
        "md": 1024px,
        "lg": 1216px,
        "xl": 1408px,
    ),
    "columns": 12,
    "container-gap": 5px 15px 30px,
    "col-gap": 5px 10px 40px,
);
$gaps: (
    "top": 10px 15px,
    "bottom": 10px 30px,
    "right": 10px 30px,
    "left": 10px 30px,
);
```
